# 联系人管理系统

一个功能完善、界面美观的联系人管理系统，支持添加、编辑、删除联系人，添加书签，管理多个联系方式，以及导入/导出Excel功能

## 功能特性

### 基本功能
- ✅ **联系人管理**：添加、编辑、删除联系人
- ✅ **联系人详情**：查看和编辑联系人详细信息
- ✅ **搜索功能**：根据姓名和联系方式搜索联系人
- ✅ **过滤功能**：按全部联系人或书签联系人过滤

### 高级功能
- ✅ **书签功能**：将重要联系人添加到书签，方便快速访问
- ✅ **多个联系方式**：支持为单个联系人添加多种联系方式（电话、邮箱、地址、微信、QQ等）
- ✅ **头像上传**：支持上传联系人头像
- ✅ **本地存储**：联系人数据保存在浏览器本地存储中，刷新页面不丢失

### 导入导出
- ✅ **Excel导出**：将所有联系人导出为Excel文件
- ✅ **Excel导入**：从Excel文件导入联系人数据

## 使用方法

### 本地运行

1. 克隆或下载项目到本地
2. 使用Python启动简单的HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 直接打开

也可以直接在浏览器中打开 `index.html` 文件，但某些功能（如导入/导出）可能受限。

## 操作指南

### 添加联系人
1. 点击右上角的「新建联系人」按钮
2. 填写联系人信息：姓名、联系方式等
3. 可以上传头像
4. 点击「保存」按钮保存联系人

### 编辑联系人
1. 在左侧联系人列表中点击要编辑的联系人
2. 修改联系人信息
3. 点击「保存」按钮更新联系人

### 删除联系人
1. 目前通过清空所有信息并保存来删除联系人
2. 后续版本将添加专门的删除按钮

### 添加书签
1. 在联系人列表中点击联系人右侧的星形图标
2. 或在联系人详情页中进行书签操作

### 搜索联系人
1. 在左侧搜索框中输入搜索关键词
2. 系统会自动过滤匹配的联系人

### 导入/导出
1. **导出**：点击右上角的「导出」按钮，将所有联系人导出为Excel文件
2. **导入**：点击右上角的「导入」按钮，选择Excel文件导入联系人

## Excel文件格式

### 导出格式

导出的Excel文件包含以下列：
- **姓名**：联系人姓名
- **书签**：是否为书签联系人（是/否）
- **备注**：联系人备注信息
- **创建时间**：联系人创建时间
- **电话1、电话2...**：电话号码
- **邮箱1、邮箱2...**：电子邮箱
- **地址1、地址2...**：实际地址
- **微信1、微信2...**：微信账号
- **QQ1、QQ2...**：QQ号码

### 导入格式

导入的Excel文件应与导出格式保持一致，至少包含「姓名」列或至少一个联系方式列。

## 开发指南

### 修改样式

样式文件 `styles.css` 使用了CSS变量，可以在文件开头的 `:root` 中修改主题颜色：

```css
:root {
    --primary-color: #4CAF50;    /* 主色调 */
    --secondary-color: #2196F3;  /* 次色调 */
    /* 其他变量... */
}
```

### 添加新功能

在 `script.js` 文件中，联系人数据结构定义如下：

```javascript
{
    id: "唯一标识符",
    name: "联系人姓名",
    avatar: "头像URL",
    methods: [
        { type: "phone", value: "电话号码" },
        { type: "email", value: "邮箱地址" }
        // 其他联系方式
    ],
    note: "备注信息",
    bookmarked: false,  // 是否为书签
    createdAt: "创建时间"
}
```

### 数据存储

联系人数据使用浏览器的本地存储（localStorage）保存，键名为 `contacts`。
